Detailed Description

This class represents region of interest - object with multiple tensors attached by multiple models (e.g., region of interest with detected face and recognized age and sex. It can be produced in a pipeline after gvainference with detection model and gvaclassify with two models. Such ROI will have 3 tensors attached)
